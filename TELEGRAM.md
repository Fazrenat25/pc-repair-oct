# 🤖 Telegram Бот для TechMaster

## Настройка бота

### 1. Создайте бота в Telegram

1. Откройте Telegram
2. Найдите бота **@BotFather**
3. Отправьте команду `/newbot`
4. Введите имя бота (например, `TechMaster Notifications`)
5. Введите username бота (должен заканчиваться на `bot`, например `techmaster_oct_bot`)
6. BotFather выдаст **токен** бота (выглядит как `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`)

### 2. Узнайте свой Telegram ID

1. Найдите бота **@userinfobot** или **@getmyid_bot**
2. Отправьте ему любое сообщение
3. Он пришлёт ваш числовой ID (например, `123456789`)

### 3. Напишите боту первым

1. Найдите своего созданного бота по username
2. Нажмите **Start** или отправьте `/start`
3. Это нужно, чтобы бот мог отправлять вам сообщения

### 4. Настройте бота в админ-панели

1. Откройте админ-панель: http://localhost:3000/admin/telegram
2. Введите **Токен бота** (из шага 1)
3. Введите **Ваш Telegram ID** (из шага 2)
4. Нажмите **"💾 Сохранить настройки"**
5. Нажмите **"📬 Отправить тест"** — должно прийти сообщение в Telegram
6. **Перезапустите сервер** для применения настроек

## Возможности бота

### 🔔 Уведомления о новых заявках

При новой заявке с сайта бот отправит сообщение:

```
🔔 Новая заявка с сайта!

📋 Заявка #abc123
👤 Клиент: Александр
📞 Телефон: +7 (999) 123-45-67
🔧 Услуга: Ремонт компьютеров
📝 Описание: Компьютер не включается
📅 Дата: 26.03.2026 14:30

#заявка #новая
```

С кнопками управления:
- **✅ В работе** — изменить статус на "В работе"
- **✔️ Выполнена** — изменить статус на "Выполнена"
- **📞 Позвонить** — показать телефон
- **👁️ Просмотр** — ссылка на заявку
- **🗑️ Удалить** — удалить заявку

### 📊 Команды бота

**/start** или **/help** — показать справку

**/stats** или кнопка **"📊 Статистика"**:
```
📊 Статистика заявок

✅ Всего: 25
🆕 Новые: 3
🟡 В работе: 5
✔️ Выполнены: 17
```

**/leads** или кнопка **"📋 Заявки"** — показать последние 5 заявок с кнопками управления

**/finance** или кнопка **"💰 Финансы"**:
```
💰 Финансовая статистика

💵 Общая выручка: 45 000 ₽
📋 Выполнено: 17
📊 Средний чек: 2 647 ₽
📅 За месяц: 25 000 ₽ (10 заявок)
📈 Рост: 15%
```

## Как это работает

1. Клиент оставляет заявку на сайте
2. Заявка сохраняется в базу данных
3. Сайт отправляет уведомление в Telegram боту
4. Бот пересылает сообщение администратору
5. Администратор может управлять заявкой прямо из Telegram

## Решение проблем

### Бот не отправляет уведомления

1. Проверьте, что бот настроен в админ-панели
2. Проверьте, что вы написали боту первым (нажали Start)
3. Проверьте логи сервера на наличие ошибок

### Кнопки не работают

1. Убедитесь, что сервер запущен
2. Проверьте, что API `/api/admin/leads` доступен
3. Перезапустите сервер

### "У вас нет доступа к боту"

Бот работает только с указанным Telegram ID. Если нужно добавить ещё администраторов, измените проверку в `src/lib/telegram.ts`.

## Безопасность

- Бот работает только с указанным Telegram ID
- Все API endpoints защищены аутентификацией NextAuth
- Токен бота хранится в `.env.local` (не коммитьте в git!)

## Обновление бота

Для изменения функциональности бота редактируйте `src/lib/telegram.ts` и перезапускайте сервер.
