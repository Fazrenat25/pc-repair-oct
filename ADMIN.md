## 📋 Структура админ-панели

### 1. Дашборд (`/admin/dashboard`)
- Общая статистика по заявкам
- Последние 5 заявок
- График заявок за последние 7 дней
- Быстрый доступ к основным разделам

### 2. Заявки (`/admin/leads`)
- Просмотр всех заявок с фильтрацией по статусам
- Изменение статуса заявки:
  - 🟢 Новая
  - 🔵 В контакте
  - 🟡 В работе
  - 🟣 Выполнена
  - 🔴 Отменена
- Добавление заметок к заявке
- Указание стоимости работ
- Удаление заявок

### 3. Услуги (`/admin/services`)
- Редактирование названий и описаний услуг
- Изменение цен
- Отметка "Популярная услуга"
- Управление порядком отображения

### 4. Отзывы (`/admin/reviews`)
- Редактирование текста отзывов
- Изменение рейтинга (звёзды)
- Отметка "Избранный отзыв" (показывается в первую очередь)
- Удаление отзывов

### 5. Настройки (`/admin/settings`)
- Заголовок сайта (Title)
- Контактный телефон
- Адрес мастерской
- Режим работы
- Стоимость выезда мастера
- Настройка бесплатной диагностики

## 🗄️ База данных

База данных SQLite хранится в файле `prisma/dev.db`.

### Просмотр данных
```bash
# Установить Prisma Studio
npm install -g prisma

# Запустить Prisma Studio
npx prisma studio
```

Prisma Studio откроется в браузере по адресу http://localhost:5555

## 📊 API Endpoints

### Заявки
- `GET /api/admin/leads` - Получить все заявки
- `PATCH /api/admin/leads` - Обновить заявку (status, notes, price)
- `DELETE /api/admin/leads?id=xxx` - Удалить заявку

### Услуги
- `GET /api/admin/services` - Получить все услуги
- `POST /api/admin/services` - Создать услугу
- `PUT /api/admin/services` - Обновить услугу
- `DELETE /api/admin/services?id=xxx` - Удалить услугу

### Отзывы
- `GET /api/admin/reviews` - Получить все отзывы
- `POST /api/admin/reviews` - Создать отзыв
- `PUT /api/admin/reviews` - Обновить отзыв
- `DELETE /api/admin/reviews?id=xxx` - Удалить отзыв

### Настройки
- `GET /api/admin/settings` - Получить все настройки
- `PUT /api/admin/settings` - Обновить настройки

## 🔐 Безопасность

### Защита роутов
- Все роуты `/admin/*` защищены аутентификацией
- API endpoints проверяют сессию пользователя
- При отсутствии сессии происходит редирект на `/admin/login`

### Смена пароля
Для смены пароля нужно:
1. Войти в базу данных через Prisma Studio
2. Найти администратора в таблице `Admin`
3. Сгенерировать новый хэш пароля:
   ```javascript
   const bcrypt = require('bcryptjs');
   const hash = await bcrypt.hash('новый_пароль', 10);
   ```
4. Обновить поле `password` в базе данных

## 🛠️ Разработка

### Добавление новой страницы
1. Создать папку в `src/app/admin/your-page/`
2. Создать `page.tsx` с компонентом
3. Добавить ссылку в навигацию в `layout.tsx`

### Добавление нового API endpoint
1. Создать файл в `src/app/api/your-endpoint/route.ts`
2. Использовать `withAdmin` для защиты
3. Добавить методы GET/POST/PUT/DELETE
