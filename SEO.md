## 📋 Что нужно сделать после запуска

### 1. Зарегистрировать сайт в поисковых системах

**Яндекс.Вебмастер:**
1. Перейти на https://webmaster.yandex.ru/
2. Добавить сайт `https://techmaster-okt.ru`
3. Подтвердить права (добавить мета-тег в `layout.tsx`)
4. Загрузить `sitemap.xml`

**Google Search Console:**
1. Перейти на https://search.google.com/search-console
2. Добавить сайт
3. Подтвердить права
4. Запросить индексирование

### 2. Настроить аналитику

**Яндекс.Метрика:**
1. Создать счётчик на https://metrika.yandex.ru/
2. Вставить ID в `.env.local`:
   ```
   NEXT_PUBLIC_YANDEX_METРИКА_ID=12345678
   ```
3. Раскомментировать код в `layout.tsx`

**Google Analytics:**
1. Создать свойство на https://analytics.google.com/
2. Вставить ID в `.env.local`:
   ```
   NEXT_PUBLIC_GA_ID=G-XXXXXXXXXX
   ```
3. Раскомментировать код в `layout.tsx`

### 3. Добавить сайт на карты

**Яндекс.Карты:**
- Добавить организацию через https://yandex.ru/sprav/

**Google My Business:**
- Создать профиль компании

**2ГИС:**
- Добавить компанию через https://2gis.ru/

### 4. Создать изображения для соцсетей

Создать файл `public/og-image.jpg` (1200×630px) с:
- Логотипом TechMaster
- Названием услуг
- Контактами

### 5. Обновить verification коды

В `layout.tsx` заменить:
```typescript
verification: {
  google: "your-google-verification-code",
  yandex: "your-yandex-verification-code",
}
```

## 📈 Рекомендации для продвижения

1. **Контент-маркетинг**
   - Вести блог с советами по ремонту
   - Публиковать кейсы сложных ремонтов
   - Отвечать на частые вопросы (FAQ)

2. **Локальное SEO**
   - Указать точный адрес и часы работы
   - Собирать отзывы клиентов
   - Добавить фото мастерской

3. **Ссылочная масса**
   - Разместить сайт в местных каталогах
   - Получить ссылки с партнёрских сайтов
   - Активность в социальных сетях

4. **Техническое SEO**
   - Следить за скоростью загрузки (PageSpeed Insights)
   - Проверять мобильную адаптацию
   - Мониторить ошибки индексации
